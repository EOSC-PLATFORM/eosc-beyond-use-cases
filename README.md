# eosc-beyond-use-cases
This repository is for code that showcase the usage of services of the EOSC Beyond EC funded project

Use cases:
* WP14 Composability-based applications via the EOSC Execution framework:
** Scholarly Communication Metadata Aggregator for EOSC Nodes: the full application includes a web user interface that interacts with the EOSC Beyond Resource Catalogue. here you can find a Jupyter Notebook that interacts with the havester service of the application. The notebook requests the harvesting of the ENES Catalogue, downloads the harvested records and perform some analysis on the XML metadata records.
